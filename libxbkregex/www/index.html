libxbkregex

A shared C++ library for FreeBSD (may work with other *BSD, untested) to make your live easier with regex.h Now its really easy to parse for regulary expressions including error-messages.

current version: V1.0.0 released 13 November 2001

[ libxbkregex-1.0.tar.gz ]

Installation:

just run make install, see demo.c for a C++ demo
